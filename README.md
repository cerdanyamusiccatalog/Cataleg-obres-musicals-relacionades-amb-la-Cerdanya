# Catàleg d'Obres Musicals Relacionades amb la Cerdanya

Conté sardanes, ballets, cançons populars, himnes i música de saló. També obres d'Antoni Degollada Barnola, Issi Fabra Bover, Antoni Monsó Lledós i Pablo Ronsó Font. La majoria d'elles provenen dels fons d'Antoni Degollada i Josep Maria Martí Terrada de l'Arxiu Comarcal de Cerdanya, de l'arxiu de Músics per la Cobla i de la web Boig Sardanista.

## Descripció

Aquest repositori conté un catàleg exhaustiu de 115 obres musicals relacionades amb la Cerdanya, ja sigui pels seus títols o pels seus autors. Les obres es classifiquen en les següents categories:

- **Sardanes (Sar)**: Danses tradicionals catalanes
- **Ballets i cançons populars (Ball)**: Obres de caire tradicional i popular
- **Altres (Alt)**: Himnes i música de saló majoritàriament
- **Antoni Degollada i Barnola (Deg)**: Obres del músic de Puigcerdà
- **Issi Fabra Bover (Fab)**: Obres del músic nascut a Puigcerdà
- **Antoni Monsó i Lledós (Mon)**: Obres del mossèn i organista
- **Pablo Ronsó i Font (Ron)**: Obres del músic nascut a Puigcerdà

## Estructura del Catàleg

El catàleg es presenta en format JSON amb els següents fitxers:

- `catalog-index.json`: Índex complet de totes les 115 obres
- `metadata-schema.json`: Esquema de metadades que defineix la estructura de cada obra
- `collections.json`: Informació sobre els fons i arxius d'origen

## Col·leccions i Arxius

Les obres provenen dels següents fons i arxius:

- **Fons Antoni Degollada Barnola** - Arxiu Comarcal de Cerdanya
- **Fons Josep Maria Martí Terrada** - Arxiu Comarcal de Cerdanya
- **Arxiu de Músics per la Cobla**
- **Base de dades Boig Sardanista**

## Com Utilitzar el Catàleg

Per accedir als fitxers JSON del catàleg:

1. Consulteu `catalog-index.json` per a la llista completa d'obres
2. Consulteu `metadata-schema.json` per a entendre l'estructura de les dades
3. Utilitzeu `collections.json` per a informació sobre els arxius d'origen
